Program Keahlian SMK Negeri 1 Wanayasa

SMK Negeri 1 Wanayasa menyelenggarakan 4 program keahlian yang telah terakreditasi dan disesuaikan dengan kebutuhan dunia kerja serta perkembangan teknologi terkini. Setiap program keahlian didukung oleh tenaga pengajar profesional, fasilitas lengkap, dan kerjasama dengan industri.


TEKNIK KOMPUTER & JARINGAN (TKJ)

Deskripsi Program:
Program keahlian TKJ mempersiapkan siswa menjadi teknisi komputer dan jaringan yang handal. Siswa akan mempelajari instalasi, konfigurasi, maintenance sistem komputer, jaringan LAN/WAN, server, dan troubleshooting perangkat IT.

Keahlian Yang Dipelajari:
✓ Sistem Komputer & Hardware
✓ Jaringan LAN/WAN & Server
✓ Administrasi Sistem Jaringan
✓ Cyber Security & Troubleshooting
✓ Cloud Computing & Virtualisasi

Peluang Karir:
Network Admin • System Admin • IT Support • Web Developer • Security Specialist


REKAYASA PERANGKAT LUNAK (RPL)

Deskripsi Program:
Program keahlian RPL fokus pada pengembangan aplikasi, website, dan game. Siswa akan belajar berbagai bahasa pemrograman, database, mobile development, dan metodologi pengembangan software modern.

Keahlian Yang Dipelajari:
✓ Programming Java, HTML, C++
✓ Web Development (HTML, CSS, JS)
✓ Mobile App Development
✓ Database Management
✓ Game Development & UI/UX

Peluang Karir:
Software Developer • Web Developer • Mobile Developer • Game Developer • UI/UX Designer


AGRIBISNIS PENGOLAHAN HASIL PERTANIAN (APHP)

Deskripsi Program:
Program keahlian APHP mengembangkan kemampuan siswa dalam mengolah hasil pertanian menjadi produk bernilai tambah. Meliputi teknologi pengolahan pangan, pengemasan, marketing produk, dan manajemen usaha agribisnis.

Keahlian Yang Dipelajari:
✓ Budidaya Tanaman Modern
✓ Teknologi Pengolahan Pangan
✓ Quality Control & Food Safety
✓ Packaging & Branding Product
✓ Agribusiness Management

Peluang Karir:
Entrepreneur Agribisnis • Food Technologist • Quality Control • Product Manager


AGRIBISNIS TERNAK RUMINANSIA (ATR)

Deskripsi Program:
Program keahlian ATR membekali siswa dengan pengetahuan dan keterampilan dalam beternak ruminansia (sapi, kambing, domba, kerbau). Meliputi manajemen kandang, pakan, kesehatan ternak, dan pengembangan usaha peternakan modern.

Keahlian Yang Dipelajari:
✓ Anatomi & Fisiologi Ternak
✓ Nutrisi & Pakan Ternak
✓ Kesehatan & Reproduksi Ternak
✓ Manajemen Kandang Modern
✓ Livestock Business Management

Peluang Karir:
Peternak Profesional • Konsultan Peternakan • Teknisi IB • Supervisor Farm
